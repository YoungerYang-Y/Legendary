# legendary

legendary的前端工程

## Project Setup

```sh
yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn run dev
```

### Compile and preview for Development

```sh
yarn run preview
```

### Type-Check, Compile and Minify for Production

```sh
yarn run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
yarn run test:unit
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn run lint
```
